# threads-farm- (redirect)

This repo exists solely to redirect traffic from the old URL to the new one.

## What happened

The original repo was named `threads-farm-` (with a trailing dash), resulting in the URL:
```
https://dantanini.github.io/threads-farm-/
```

After the site went live and links were shared on social media, we renamed the repo to `threads-farm` for a cleaner URL. However, GitHub Pages does not automatically redirect project page URLs after a rename.

Since the old URL was already shared publicly, we created this repo to serve a redirect page, ensuring existing links continue to work.

## Redirect target

All traffic is redirected to: https://dantanini.github.io/threads-farm/

## Lesson learned

Choose your repo name carefully before going live — GitHub Pages project sites don't auto-redirect after rename.
